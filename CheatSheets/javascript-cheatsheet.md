## 1. Declaring Variables

| Declaration | Code            | Notes                                     |
| ----------- | --------------- | ----------------------------------------- |
| var         | ` var a = 1;`   | Can be updated and re-declared.           |
| let         | ` let a = 1;`   | Can be updated but cannot be re-declared. |
| const       | ` const a = 1;` | Cannot be updated or re-declared.         |
